Os H1, H2, H3, H4, H5 e H6 são elementos de títulos usados em HTML, e você pode chamá-los de "cabeçalhos" ou "títulos". Cada um tem uma função hierárquica:

  • H3: Subtítulo do H2, mais específico.
  • H4: Subtítulo do H3, mais detalhado.
  • H5: Subtítulo do H4.
  • H6: Subtítulo do H5, geralmente o nível mais baixo.

Esses elementos são usados para estruturar o conteúdo de forma hierárquica, tanto para usabilidade como para SEO (otimização para mecanismos de busca). CSS: Estilizando a Web

O CSS (Cascading Style Sheets) é a linguagem de estilos utilizada para definir a aparência das páginas web. Enquanto o HTML estrutura o conteúdo, o CSS controla como esses elementos serão exibidos no navegador. Com o CSS, você pode alterar cores, fontes, espaçamentos, alinhamentos e outros aspectos visuais de uma página, criando um design atraente e funcional.

Principais Funcionalidades do CSS

O CSS oferece uma grande flexibilidade para estilizar páginas web, e algumas de suas funcionalidades principais incluem:

  • Seletores: Permitem que você defina regras para elementos específicos, como uma classe, id ou tipo de tag.
  • Propriedades: Usadas para definir as características do elemento, como cor, fonte, bordas, margens e alinhamento.
  • Layouts Responsivos: Técnicas como o uso de media queries permitem que o design da página se ajuste de forma eficiente a diferentes tamanhos de tela, garantindo que seu site seja bem exibido tanto em dispositivos móveis quanto em desktops.

Com o CSS, a personalização visual de um site é quase ilimitada, e é essencial para quem deseja criar uma experiência de usuário agradável e moderna.

Usando <b> (apenas negrito):

html
<p>Os <b>seletores</b> são usados para escolher os elementos HTML que você deseja estilizar.</p>

Embora ambos resultem no mesmo efeito visual (texto em negrito), a tag <strong> é a melhor escolha quando você quer dar um significado adicional ao texto, como dar ênfase ou destacar algo importante, enquanto o <b> é mais para fins de estilo.

JavaScript: Tornando a Web Interativa

O JavaScript é uma das linguagens de programação mais importantes no desenvolvimento web. Ele permite adicionar interatividade a páginas web, tornando-as dinâmicas e responsivas. Ao contrário do HTML e CSS, que são usados para estruturar e estilizar, o JavaScript é usado para controlar o comportamento e as ações de uma página.

Principais Funcionalidades do JavaScript

JavaScript pode ser usado para uma série de funcionalidades, incluindo:

  • Manipulação de elementos HTML: JavaScript permite alterar o conteúdo e o estilo de elementos HTML em tempo real, criando uma experiência dinâmica.
  • Eventos: Você pode usar JavaScript para responder a ações dos usuários, como cliques de botão ou movimento do mouse.
Exemplo Básico de JavaScript

Aqui está um exemplo simples de como usar JavaScript para mostrar uma mensagem quando um botão é clicado:

html
<button onclick="alert('Olá, mundo!')">Clique-me</button>

No código acima, o JavaScript é usado para exibir uma mensagem quando o usuário clica no botão.
